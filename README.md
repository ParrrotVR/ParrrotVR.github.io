# Mysmic Portfolio

The source for the Mysmic portfolio at [parrrotvr.github.io](https://parrrotvr.github.io/) — a monochrome portfolio for browser engineering, game preservation, and security research.

Built with React, Vite, and selected open-source visual patterns from [React Bits](https://reactbits.dev/).

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
